# Linux DevOps Practice Tasks

This repository contains hands-on Linux practice tasks completed on AWS EC2 Ubuntu 26.04 LTS instance.

## Environment
- **OS:** Ubuntu 26.04 LTS
- **Platform:** AWS EC2 t2.micro
- **Region:** Asia Pacific (Mumbai)

## Tasks Completed

### Task 1 - File System Structure & Navigation
- Explored root directory structure using pwd, ls, ls -l
- Navigated /etc, /var, /home directories
- Listed contents of /bin, /boot, /dev, /proc, /tmp
- Practiced absolute and relative path navigation
- Created and removed files and directories

### Task 2 - Wildcards & Soft/Hard Links
- Used wildcards (*, ?, []) to filter files
- Created and tested hard links vs soft links
- Understood inode numbers and link behavior
- Tested link behavior after original file deletion

### Task 3 - Find, Locate & Copy Commands
- Created nested directory structure for practice
- Used find command with -name and -type flags
- Installed plocate and used locate command
- Copied files between directories using cp
- Copied entire directories recursively using cp -r
- Created backup using cp -r

### Task 4 - Find, Locate & Copy (Resubmission)
- Corrected find command syntax
- Demonstrated find vs locate difference
- Verified all copy operations with ls output
- Captured complete command history
